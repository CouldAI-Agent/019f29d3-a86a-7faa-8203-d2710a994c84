# السيرة الذاتية الاحترافية

تطبيق سيرة ذاتية (CV) احترافي ومبني باستخدام إطار عمل Flutter. يتميز التطبيق بتصميم متجاوب يعمل بكفاءة على شاشات الهواتف المحمولة بالإضافة إلى شاشات الحواسيب المكتبية والويب. تم تصميم التطبيق لدعم اللغة العربية بشكل كامل (من اليمين إلى اليسار RTL).

## المميزات
- **تصميم متجاوب**: يتكيف التصميم تلقائياً بناءً على حجم الشاشة. على الهواتف يظهر كعمود واحد، بينما على الشاشات الكبيرة يظهر كقسمين (شريط جانبي ومحتوى رئيسي).
- **دعم اللغة العربية**: تم ضبط إعدادات الاتجاه في التطبيق لعرض النصوص من اليمين إلى اليسار (RTL).
- **أقسام متكاملة**: يتضمن التطبيق أقسام للمعلومات الشخصية، وطرق التواصل، والمهارات، والخبرات المهنية، والتعليم، والمشاريع البارزة.
- **كود منظم**: تم بناء واجهة المستخدم بطريقة نظيفة وقابلة للتعديل والتوسع لتمكين المستخدمين من تغيير بياناتهم بسهولة.

## كيفية التشغيل
1. تأكد من تثبيت [Flutter](https://flutter.dev/docs/get-started/install) على جهازك.
2. قم بفتح المشروع في محرر الأكواد المفضل لديك (مثل VS Code أو Android Studio).
3. احصل على الحزم اللازمة عبر تنفيذ الأمر:
   ```bash
   flutter pub get
   ```
4. قم بتشغيل التطبيق على الهاتف أو المتصفح باستخدام:
   ```bash
   flutter run
   ```
